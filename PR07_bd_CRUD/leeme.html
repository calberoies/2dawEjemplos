Ejemplo de CRUD con PDO

Este ejemplo muestra las acciones típicas de base de datos (CRUD en inglés):
  • C-Create
  • R-Retrieve
  • U-Update
  • D-Delete
El acceso a la Base de datos se implementa con PDO

Incluye:
  • Ejecución de consultas preparadas, con parámetros
  • Lectura de datos con distintos modos PDO::FETCH_XXXX
  • Desplegable de datos relacionados por clave externa
  • Login/logout de usuarios
  • Filtro de acciones si el usuario no está logueado
  • Transacciones
